# where42 frontend

이 레포지토리는 ["어디있니"](http://where42.kr/) 웹 서비스의 프론트엔드 부분을 관리합니다.

## 기능

- **카뎃의 위치 정보**: 친구로 등록하거나 검색한 카뎃이 클러스터에 있는지 위치정보를 제공합니다

## 실행 방법

- node, git 설치가 필요합니다

```bash
git clone https://github.com/42Where/where42_front_v2.git

cd where42_front_v2

npm install

npm run dev
```

## 기술 스택

- Next.js
- React
- Zustand
- TypeScript
- SASS
- Storybook
